How we calculate

Mfg landed = Base mfg + TPE resin + Colorant + Mesh bag + Box + Pallet freight (+ plate fee/pair if brand pays).
Fully loaded = Mfg landed + 3PL handling + Ship to customer.

Base mfg ($5.09) — per-pair cost including packaging and logo at factory.
TPE resin ($1.01) — 0.388 lbs (Large pair) × $2.60/lb mid-tier Kraiburg TC2GPN. Tiers: 1,980 lbs = $2.87, 4,690 lbs = $2.60, 9,920 lbs = $2.42.
Colorant ($0.31) — 0.388 lbs × let-down % × $/lb. Default: 4% at $20/lb. Range: $12–$20/lb.
Mesh bag ($0.56) — 1 per pair. Quote: $0.56/pc (29.5×20.3cm).
Box ($0.60) — 1 kraft mailer per pair (conservative). At 2–3 pairs/box: $0.20–$0.30.
Pallet freight ($0.23) — ~$350/pallet ÷ ~1,500 pairs.

— 3PL Fulfillment (EFS) —
3PL handling ($1.95) — pick/pack $0.70 + order processing $1.25. Per EFS invoice Jun 2026.
Ship to customer ($8.20) — avg USPS/FedEx postage. Range $7.13–$16.25. Set $0 if customer pays.
Storage (~$14.89/wk) and support ($25/wk) are fixed overhead, not included per-pair.
