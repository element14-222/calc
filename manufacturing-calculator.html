How we calculate

Brand margin % = (Sell-in price − Fully loaded cost) / Sell-in price.

Mfg landed = Base mfg + TPE resin + Colorant + Mesh bag + Box + Shipping (+ plate fee spread if brand pays).
Fully loaded = Mfg landed + 3PL handling + Ship to customer.

Base mfg ($5.09) — per-pair manufacturing cost including packaging and logo at factory.
Mesh bag ($0.56) — 1 mesh bag per pair. Quote: $0.56/pc (29.5×20.3cm, black or white).
Box ($0.60) — 1 kraft mailer box per pair (conservative). Box $0.34 + sea shipping $0.26 = $0.60 landed. At 2–3 pairs/box, cost drops to $0.20–$0.30/pair.
TPE resin ($1.01) — 0.388 lbs (Large pair) × $2.60/lb mid-tier Kraiburg TC2GPN. Volume tiers: 1,980 lbs = $2.87/lb, 4,690 lbs = $2.60/lb, 9,920 lbs = $2.42/lb.
Colorant — 0.388 lbs (Large pair weight) × let-down % × $/lb. Default: 4% let-down at $20/lb = $0.31/pair. Range: $12–$20/lb.
Shipping ($0.23) — pallet freight: ~$350/pallet ÷ ~1,500 pairs = $0.23/pair.

— 3PL Fulfillment (EFS) —
3PL handling ($1.95) — pick/pack $0.70/item + order processing $1.25/order. Based on EFS invoice Jun 2026.
Ship to customer ($8.20) — avg postage via USPS Ground Advantage / FedEx SmartPost. Range $7.13–$16.25. Set to $0 if customer pays shipping.
Storage (~$14.89/wk) and support ($25/wk) are fixed overhead, not included per-pair.
